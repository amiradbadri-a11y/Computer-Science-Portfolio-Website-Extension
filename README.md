# Personal Portfolio Website

## Description

This is a simple personal portfolio website created using HTML and CSS.  

It includes two pages that introduce me and provide basic information about my studies and interests in computer science and AI.

## Pages

- Home page (index.html)

- About page (second.html)

## Features

- Simple and clean design

- Two connected web pages

- Internal CSS styling

- Navigation links between pages

## How to Run

1. Download or clone this repository  

2. Open the folder  

3. Open `index.html` in any web browser  

## Technologies Used

- HTML

- CSS
